```math
\dot{v}_t=f(x_t)-h(x_t)\int_0^t \theta(t-\tau) h(x_\tau)v_\tau d\tau +h(x_t)R_t
```

```math
h(x)= \frac{\langle \dot{v}_0-f(x_0),\dot{v}_0 |x_0=x \rangle}{\langle v_0,v_0 \rangle}
```
